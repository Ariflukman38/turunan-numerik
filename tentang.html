📌 Deskripsi Aplikasi

Aplikasi Kalkulator Metode Numerik merupakan aplikasi berbasis web yang dirancang untuk membantu mahasiswa dalam menghitung nilai turunan suatu fungsi menggunakan metode pendekatan numerik. Aplikasi ini mendukung tiga metode utama yaitu Forward Difference (Hampiran Selisih Maju), Backward Difference (Hampiran Selisih Mundur), dan Central Difference (Hampiran Selisih Pusat).

Aplikasi ini diharapkan dapat mempermudah proses perhitungan turunan numerik, mengurangi kesalahan perhitungan manual, serta menjadi media pembelajaran interaktif bagi mahasiswa Teknik Informatika.

Aplikasi ini dibuat untuk memenuhi tugas project akhir pada Mata Kuliah Metode Numerik Program Studi Teknik Informatika Angkatan 24 Universitas Perjuangan Tasikmalaya.

🎯 Tujuan

  1. Membuat aplikasi kalkulator metode numerik yang mudah digunakan.
  2. Membantu mahasiswa memahami proses perhitungan turunan numerik.
  3. Mengimplementasikan algoritma Forward, Backward, dan Central Difference.
  4. Menyediakan langkah-langkah perhitungan secara detail.
