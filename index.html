Kalkulator Turunan Fungsi Hampiran Selisih

Aplikasi perhitungan berbasis Web untuk menghitung turunan fungsi menggunakan metode numerik Hampiran Selisih seperti Forward Difference, Backward Difference, dan Central Difference.

📌 Tentang Aplikasi

Metode numerik adalah teknik penyelesaian masalah matematika menggunakan pendekatan komputasi. Aplikasi ini mendukung metode turunan numerik menggunakan selisih maju, selisih mundur, dan selisih pusat.

⚙️ Fitur Utama

  • Perhitungan Turunan Numerik
  • Forward Difference (Selisih Maju)
  • Backward Difference (Selisih Mundur)
  • Central Difference (Selisih Pusat)
  • Langkah Penyelesaian Otomatis

📚 Materi Dasar Hampiran Selisih

➜ Forward Difference (Selisih Maju)

Menggunakan nilai fungsi pada titik saat ini dan titik setelahnya. Biasanya digunakan pada awal tabel data.

f'(x₀) = [f(x₀+h) - f(x₀)] / h
f'(x₀) = (f₁ - f₀) / h

← Backward Difference (Selisih Mundur)

Menggunakan nilai fungsi pada titik saat ini dan titik sebelumnya. Cocok digunakan pada bagian akhir tabel data.

f'(x₀) = [f(x₀) - f(x₀-h)] / h
f'(x₀) = (f₀ - f₋₁) / h

↔ Central Difference (Selisih Pusat)

Menggunakan titik sebelum dan sesudah titik yang dihitung. Umumnya menghasilkan galat yang lebih kecil.

f'(x₀) = [f(x₀+h) − f(x₀−h)] / 2h
f'(x₀) = (f₁ − f₋₁) / 2h
