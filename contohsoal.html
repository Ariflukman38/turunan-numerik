Teori Hampiran Selisih

Forward, Backward dan Central Difference

Pengertian

Hampiran Selisih (Finite Difference) merupakan metode numerik untuk memperkirakan nilai turunan suatu fungsi dengan menggunakan data fungsi pada titik-titik tertentu. Metode ini digunakan ketika turunan analitik sulit diperoleh atau hanya tersedia data tabel. 

Selisih Maju (Forward Difference)

f'(x) ≈ [f(x+h)-f(x)] / h

Selisih Mundur (Backward Difference)

f'(x) ≈ [f(x)-f(x-h)] / h

Selisih Pusat (Central Difference)

f'(x) ≈ [f(x+h)-f(x-h)] / 2h

Turunan Kedua Selisih Pusat

f''(x) ≈ [f(x+h)-2f(x)+f(x-h)] / h²
∑

Teori Deret Taylor

Pendekatan fungsi menggunakan ekspansi Taylor

Pengertian

Deret Taylor digunakan untuk mendekati nilai fungsi di sekitar suatu titik menggunakan informasi turunan fungsi pada titik tersebut. Semakin tinggi orde Taylor yang digunakan, maka hasil pendekatan akan semakin akurat. 


f(x+h) = f(x) + hf'(x) + (h²/2!)f''(x) + (h³/3!)f'''(x) + (h⁴/4!)f''''(x)+...
≈

Teori Interpolasi Newton

Newton Forward Interpolation

Pengertian

Interpolasi digunakan untuk memperkirakan nilai fungsi pada titik yang tidak terdapat dalam tabel data. Metode Newton Forward memanfaatkan tabel selisih maju untuk membentuk polinom interpolasi. 


P(x) = f₀ + sΔf₀ + [s(s-1)/2!]Δ²f₀ + [s(s-1)(s-2)/3!]Δ³f₀ + ...

dengan

s = (x-x₀)/h
①

Mode Fungsi - Hampiran Selisih Pusat

Menghitung Turunan Pertama Menggunakan Central Difference

Diketahui

Fungsi :f(x)=x³+2x²−x+1

Nilai x = 2
Nilai h = 0.1
Metode = Central Difference
Turunan yang dicari = f'(x)

Langkah 1 - Gunakan Rumus Selisih Pusat


f'(x) ≈ [f(x+h)-f(x-h)] / 2h

Langkah 2 - Hitung Nilai Fungsi


f(2.1) = (2.1)³ + 2(2.1)² - 2.1 + 1 = 17.081000

f(1.9) = (1.9)³ + 2(1.9)² - 1.9 + 1 = 12.299000

Langkah 3 - Substitusi ke Rumus


f'(2) = [17.081000 - 12.299000] / (2 × 0.1) 
      = 4.782000 / 0.2
      = 23.910000
Hasil Turunan Pertama 23.910000
